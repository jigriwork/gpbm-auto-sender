# Mobile UI

The private app now includes mobile-first behavior:

- Sticky mobile header.
- Drawer navigation.
- Stacked content cards.
- Bill tables become mobile cards.
- Filter controls stack on small screens.
- Forms fit screen width.
- Buttons are touch-friendly.
